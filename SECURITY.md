# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities via [GitHub Security Advisories](https://github.com/pradeepmouli/x-to-zod/security/advisories/new) rather than filing a public issue.

We aim to acknowledge reports within 48 hours and provide a timeline for resolution within one week.

## Supported Versions

Only the latest minor version receives security updates while this project is pre-1.0.
